# Task Management System

A simple **Task Management Web Application** built using **Python Flask, HTML, CSS, and MYSQL**.

##  Features

* Add new tasks
* Edit tasks
* Delete tasks
* View all tasks
* Simple and clean interface

## 🛠 Technologies Used

* Python
* Flask
* HTML
* CSS
* MYSQL

## How to Run the Project

1. Clone the repository

```
 git clone https://github.com/arpitanayak29/task-management-system
```

2. Go to project folder

```
cd task-management-system
```

3. Install Flask

```
pip install flask
```

4. Run the application

```
python app.py
```

5. Open in browser

```
http://127.0.0.1:5000
```
